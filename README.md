# my-cogs

A collection of custom [Red-DiscordBot](https://github.com/Cog-Creators/Red-DiscordBot) cogs.

## Installation

```
[p]repo add my-cogs https://github.com/<your-username>/<your-repo>
[p]cog install my-cogs <cog-name>
[p]load <cog-name>
```

## Cogs

| Cog | Description |
|---|---|
| [autokick](./autokick) | Kicks members who never receive any of one or more designated "verified" roles within N days of joining. Built for servers using a ticket-based approval flow. |
